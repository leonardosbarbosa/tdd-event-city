# tdd-event-city

# Sobre esse projeto
Um sistema de eventos e cidades com relação N-1 entre eles. Foi criado com o objetivo de usar na prática a metodologia TDD (Test Driven Development).

# Tecnologias utilizadas
- Java 11
- Spring Boot (2.4.5)
- JPA / Hibernate
- Maven
- H2 database
